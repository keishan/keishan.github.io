The Periodic Table consists of 118 different elements with different purposes.

About the Periodic Table

  • Carbon is unique in that it is known to form up to 10 million different compounds.
  • The only letter not in the periodic table is the letter J.
  • Francium is the rarest element on earth.

Top 3 facts about the Periodic Table

  1. There are 90 elements on the periodic table that occur in nature. All of the other elements are strictly man-made.
  2. Most of the elements on the periodic table are metals. The alkali metals, alkaline earths, basic metals, transition metals, lanthanides and actinides all are groups of metals.
  3. Technetium was the first element to be made artificially.
